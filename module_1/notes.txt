существует 3 версии Java - SE (Standard Edition), EE (Enterprise Edition) и ME (mobile edition)
Перевод программы, написанной на java происходит следующим образом - исходный файл через компилятор javac (java compilator)в byte code а дальше интерпретируется через JVM
(java virtual machine) в машинный код.


Типы данных:

целые числа:
    bytes - целое число - 1 байт
    short - короткое число - 2 байта
    int - обычное число - 4 байта
    long - длинное число - 8 байт

числа с плаваюзей точкой:
    float - 4 байта
    double - 8 байт

символы:
    char - 2 байта - unicode

boolean numbers:
    TRUE & FALSE

final - создание некоторой константы
